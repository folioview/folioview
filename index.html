Custom Price Proxy (optional)
If price fetching is slow or blocked - most common when opening the file directly on a phone - you can route requests through your own free Cloudflare Worker. It becomes the first-choice route, with the built-in free proxies as fallback. Copy the Worker code below, deploy it on Cloudflare (steps included, free tier is plenty), then paste its URL here.
How to deploy (free, ~5 minutes)
  1. Click Copy Worker code below.
  2. Sign up / log in at dash.cloudflare.com (free, no card).
  3. In the left menu go to Build > Compute (Workers & Pages).
  4. Click Create application > Start with Hello World > name it (e.g. folionaire-price-updates) > Deploy.
  5. Click Edit code, delete the sample, paste the copied code, then Deploy.
  6. Copy the Worker URL (https://name.account.workers.dev), paste it above, hit Test, then Save.
The code only allows Folionaire's data sources, so nobody can misuse your Worker as an open proxy. The free tier (100k requests/day) is far more than one person needs.

About
Folionaire v1.0 - Personal portfolio tracker

Not financial or tax advice
Folionaire is a personal tracking and information tool only - it is not financial, investment, or tax advice. The CGT and tax figures are estimates based on general Australian rules and may not reflect your circumstances. Always verify with a registered tax agent or licensed financial adviser before making decisions or lodging a return.

Your data stays with you
Everything is saved locally in your browser - your holdings are never uploaded to a server or cloud account, and no account is required. Export to JSON any time to back up your data or move it to another device.

API dependency
Live prices are powered by CoinGecko and Yahoo Finance. News & Sentiment uses Google News RSS with a Yahoo Finance RSS fallback, fetched through a chain of free CORS proxies (or your own Marketaux, Finnhub, or Alpha Vantage key if you add one). If these services change, go down, or become unavailable, any feature that relies on them - price fetching, Investment Overview, Terminal data, News & Sentiment - will stop working. If that happens, a future update will look at replacing them with alternative free APIs. In the meantime, Folionaire still works fully as a transaction ledger, CGT calculator, and portfolio tracker using your last known prices or manual entries.

Running locally
For best results, serve this file from a local web server rather than opening it directly as a file in your browser. Opening via file:// can trigger CORS restrictions that block API calls. A simple local server (e.g. Python's http.server, Node's serve, or Termux on Android) resolves this. All data stays on your device regardless.
